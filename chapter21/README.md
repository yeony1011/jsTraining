# One Day One JavaScript 17

다음 요구사항에 따라 **카운트다운 타이머(Countdown Timer)**를 구현하시오.
유효한 날짜(예:"2025-12-31 24:00:00")를 입력받아,
현재 시점으로부터 해당 날짜까지 남은 **일(day), 시(hour), 분(minute), 초(second)**를
실시간으로 표시하는 카운트다운 타이머를 구현한다.

상세 요구사항
- 모든 동작은 순수 자바스크립트로 구현한다.
- 날짜 입력은 ```<input>``` 또는 ```prompt()```를 통해 문자열로 입력받는다.
- 입력받은 날짜는 이전 과제(날짜 유효성 검사기) 로직을 참고하여 유효성 검사를 통과해야 한다.
- 입력한 날짜가 현재 시각보다 이전인 경우, "이미 지난 날짜입니다." 메시지를 표시하고 종료한다.
- 남은 시간이 0이 되면 "카운트다운이 종료되었습니다." 메시지를 표시한다.
- 검색을 통해 스스로 해결하며, AI를 활용해 코드를 생성하거나 회신받는 행위는 금지한다.

# STUDY
- new Data(비교날짜) < new Date() : 입력날짜가 현재 시간보다 과거인지 체크 가능
- setHour(0,0,0,0) : 당일 00:00:00 기준으로 비교할 때 사용
- getTime() : 정확한 밀리초 단위 비교 가능
