# One Day One JavaScript 6

다음 요구사항에 따라 **랜덤 난수 생성기(Random Code Generator)**를 구현하시오.
화면에는 “난수 생성” 버튼이 있으며, 버튼을 클릭할 때마다 알파벳 대문자와 숫자가 조합된 형태의 랜덤 코드를 생성하고 화면에 출력한다.
코드 형식과 규칙은 A00A0000와 같이 출력한다.

## 상세 요구사항
- 모든 동작은 순수 자바스크립트로 구현한다.
- 화면에는 “난수 생성” 버튼(```<button>```)과 결과를 표시할 ```<div>``` 영역이 존재한다.
- “난수 생성” 버튼 클릭 시, A00A0000 형식의 문자열을 랜덤하게 생성한다. 첫 번째와 네 번째 자리는 A~Z 사이의 대문자 알파벳 나머지 자리는 0~9 사이의 숫자로 구성한다.
- 별도 옵션, 또는 파라미터를 통해 알파벳의 자리수와 숫자의 배치가 유동적으로 변경될 수 있도록 한다. (예 : 영문 대/소문자 갯수, 난수 길이 / 영문 위치 등)
- 생성된 난수는 결과 영역에 출력하며, 클릭할 때마다 새로운 난수가 생성되어 기존 값이 갱신된다.
- 검색을 통해 스스로 해결하며, AI를 활용해 코드를 생성하거나 회신받는 행위는 금지한다.

## STUDY
- Math.random() : 0 ~ 1 사이의 임의의 수를 반환하는 메서드
- Math.floor() : 소수점 이하 버림
- Math.ceil() : 소수점 이하 올림
- Math.round() : 소수점 이하 반올림
- fromCharCode() : 아스키코드표의 숫자에 해당하는 문자열로 변환
