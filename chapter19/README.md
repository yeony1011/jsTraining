# One Day One JavaScript 15

다음 요구사항에 따라 **엑셀 파일 편집 및 다운로드(Excel Edit & Download)**를 구현하시오.
화면에서 사용자가 엑셀 파일을 첨부하면, 데이터를 HTML 테이블로 표시하고 편집 가능하도록 한다.
편집한 내용은 편집 후 엑셀 파일로 다시 다운로드할 수 있어야 한다.

상세 요구사항
- 모든 동작은 순수 자바스크립트로 구현한다.
- 페이지에는 <input type="file" id="excelInput">, <div id="tableContainer">, <button id="downloadBtn">다운로드</button>가 존재한다.
- 엑셀 파일을 선택하면, 파일의 내용을 HTML <table>로 변환하고, <td>는 contenteditable="true" 속성을 사용해 편집 가능하게 표시한다.
- “다운로드” 버튼 클릭 시, 변경된 편집 데이터를 기반으로 새로운 엑셀 파일을 생성하고 다운로드할 수 있어야 한다.
- 검색을 통해 스스로 해결하며, AI를 활용해 코드를 생성하거나 회신받는 행위는 금지한다.
