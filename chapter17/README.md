# One Day One JavaScript 13

다음 요구사항에 따라 **이미지 미리보기(Image Preview)**를 구현하시오.
화면에는 이미지 파일 선택 입력창과, 선택된 이미지들을 표시할 썸네일 영역이 존재한다.
사용자가 이미지를 선택하면, 선택된 이미지가 썸네일 형태로 화면에 표시되고, 이미지 배열이 관리된다.
배열을 기반으로 정렬, 삭제, 랜덤 배치 등 기능을 구현할 수 있도록 한다.

상세 요구사항
- 모든 동작은 순수 자바스크립트로 구현한다.
- 화면에는 다음과 같은 요소가 존재한다. (```<input type="file" id="imageInput" multiple> <div id="preview"></div> <div id="list"></div>```)
- 이미지를 첨부하면, list 영역에 썸네일 형태로 이미지가를 노출 한다.
- 썸네일 이미지를 클릭하면, preview영역에 이미지를 노출 한다.
- DOM 업데이트는 기존 요소를 제거하지 않고, appendChild 등의 방법으로 처리한다.
- 여러 이미지를 선택해도 배열 순서와 DOM 순서를 동기화할 수 있어야 한다.
- 각 썸네일의 삭제 버튼 클릭 시 이미지 삭제
- 검색을 통해 스스로 해결하며, AI를 활용해 코드를 생성하거나 회신받는 행위는 금지한다.
