# One Day One JavaScript 9

다음 요구사항에 따라 **HTML 엘리먼트 랜덤 배치기(Random Element Shuffler)**를 구현하시오.
화면에는 기본적으로 ```<div>``` 엘리먼트들이 존재하며, 각 엘리먼트에는 콘텐츠가 표시된다.
사용자가 특정 동작(예: 버튼 클릭)을 수행하면, 화면에 있는 엘리먼트의 순서가 랜덤하게 재배치된다.

## 상세 요구사항
- 모든 동작은 순수 자바스크립트로 구현한다.
- 초기 화면에는 예시와 같이 콘텐츠가 포함된 ```<div>``` 요소가 존재한다. (예 : ```<div>a</div> <div>b</div>```)
- 페이지 로드 시, 화면 상에 표시된 ```<div>``` 요소들의 순서를 무작위로 섞어 다시 배치한다.
- 요소의 순서 변경 시, 콘텐츠(a, b)는 그대로 유지되어야 한다.
- 기존 요소를 제거하고 새로 생성하는 방식이 아니라, DOM 조작을 통해 순서만 변경한다.
- 검색을 통해 스스로 해결하며, AI를 활용해 코드를 생성하거나 회신받는 행위는 금지한다.

- 이전 ‘HTML 엘리먼트 랜덤 배치기’ 과제와 동일하며, ```<div>``` 요소가 여러 개일 경우에도 동작하도록 일반화, 다른 태그(```<li>```, ```<span>```, ```<p>``` 등)에도 적용 가능하도록 구현한다.
