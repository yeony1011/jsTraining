화면에는 엑셀 파일 선택 입력창과, 선택된 파일의 내용을 표시할 미리보기 영역이 존재한다.
엑셀 파일을 첨부하면, 선택된 데이터가 화면에 표시된다

상세 요구사항
- 모든 동작은 순수 자바스크립트로 구현한다.
- 화면에는 다음과 같은 요소가 존재한다. (```<input type="file" id="file" accept=".xls,.xlsx,.csv">``` ```<div id="preview"></div>```)
- 엑셀 파일을 선택하면, 선택된 파일의 내용을 테이블 또는 div 기반의 미리보기로 화면에 표시한다.
- 검색을 통해 스스로 해결하며, AI를 활용해 코드를 생성하거나 회신받는 행위는 금지한다.
