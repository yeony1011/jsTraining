# One Day One JavaScript 11

다음 요구사항에 따라 **HTML 엘리먼트 랜덤 배치기(Random Element Shuffler)**를 구현하시오.
화면에는 여러 콘텐츠 ```<div>```와 해당 콘텐츠를 제어할 ```<button>``` 탭이 존재한다.
페이지가 로드될 때, 콘텐츠와 탭의 순서는 랜덤하게 섞이되, 각 버튼과 매칭되는 콘텐츠는 같은 순서로 유지되며, 1개의 콘텐츠만 노출된다.
탭 버튼을 클릭하면 해당 콘텐츠가 표시되는 형태의 탭으로 구성한다.

상세 요구사항
- 모든 동작은 순수 자바스크립트로 구현한다.
- 초기 화면에는 ```<div>```와 ```<button>``` 요소가 존재한다.
- 페이지 로드시, ```<div>``` 콘텐츠와 ```<button>``` 탭의 순서를 같은 기준으로 랜덤하게 섞고 하나의 컨텐츠를 노출한다.
- 요소의 순서 변경 시, 콘텐츠와 버튼 매칭 관계는 그대로 유지되어야 한다.
- 기존 요소를 제거하고 새로 생성하는 방식이 아니라, DOM 조작을 통해 순서만 변경한다.
- ```<div>```와 ```<button>```이 여러 개일 경우에도 동작하도록 일반화한다.
- button 클릭시 매칭된 컨텐츠를 보여주는 탭으로 구성한다
- 검색을 통해 스스로 해결하며, AI를 활용해 코드를 생성하거나 회신받는 행위는 금지한다.
